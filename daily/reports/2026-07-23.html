S1 ① 腾讯混元发布 Hyra-1.0：递归自我改进的科研智能体

来源：IT之家 / 凤凰网科技，07-21

轻量 Harness + 生产者-消费者异步流水线：Context Agent 维护「经验库（Experience Bank）」持续供给灵感上下文，多个 Proposal Agent 在独立沙盒中执行并评分；双层循环让评估器与方案共进化。战绩：55 个数学开放问题中 29 个刷新历史最好结果；NanoGPT Speedrun 达到 3.28 validation loss 缩短至 76.4 秒；IBM Q20 量子路由效率较经典 SABRE 提升 44.4%；PARP1 抑制剂候选分子联合评分超过已上市药物 olaparib。对标 AlphaEvolve，但强调「评估器也会进化」。

点评： RSI（递归自我改进）今年从 DeepMind 独角戏变成集团军——Karpathy 的 autoresearch、Together 的 kissing number、现在腾讯正式下场。Hyra 真正的新意不在「会做题」，而在把经验显式资产化：成功与失败都入库，下一轮探索吃上一轮的复利。 对你： 这套 Experience Bank 与 MemoryNexus 几乎同构——Hyra 证明了「经验库 + 沙盒验证 + 评估器共进化」闭环在科研级任务上能跑出 SOTA，等于给 Agent 记忆系统做了一次大规模背书。建议读它的开源成果仓库，重点看经验条目如何表示、如何检索、如何防止评估器被 hack。12

S1 ② DeepSeek V4 正式版倒计时：峰谷定价落地，旧 API 别名明日退役

来源：36氪，07-20 / 07-22

V4 正式版官宣 7 月内上线（昨日信源称已推迟至本月底，落地进入读秒），GA 同步引入峰谷定价：每日 9:00–12:00、14:00–18:00（UTC+8）高峰时段 API 价格为平时 2 倍。V4-Pro 输出平时 0.87 美元/百万 token（高峰 1.74），V4-Flash 输出平时 0.28 美元；缓存命中的输入价依然极低（Flash 命中 0.0028 美元）。deepseek-chat 与 deepseek-reasoner 两个旧模型名 7 月 24 日正式下线。预览版官方自测 SWE-bench Verified 距 Claude Opus 4.6 Max 仅 0.2 个百分点，价格约为对方 1/7。

点评： 「价格屠夫」第一次给算力装上计价器，这是行业信号——推理需求已从「抢供给」进入「调峰」阶段，逻辑和电网一模一样。极低的缓存命中价等于官方明示：批处理、评测、数据生成请挪到夜里。 对你： 若 MemoryNexus 或你的 Agent 链路白天长跑，明天起账单直接翻倍。今天就做两件事：①把代码里的旧模型别名迁到新名字；②给调度器加「峰谷感知」，批量任务排队到 18 点后。34

S1 ③ Anthropic 15 亿美元版权和解获法院最终批准

来源：Ars Technica / TechCrunch，07-20/21

法官 Martínez-Olguín 批准美国史上最大版权和解：覆盖约 50 万部作品、每部约 3,000 美元，91% 作者与出版方已提交申领，仅 350 人退出；律师费从申请的 1.87 亿美元砍至约 1.01 亿。判例逻辑不变：合法获取的书籍用于训练属合理使用，赔的是从 LibGen 等盗版站下载的 700 万册。

点评： 这笔钱真正定价的不是「训练」，而是「来路不正」。和解意味着不构成约束性先例——OpenAI、Meta、Google、Midjourney 的案子各打各的——但「每部作品 3,000 美元」从此成为所有谈判桌上的锚。 对你： 做数据湖的该记住：数据资产的价值里现在包含「来源凭证」。表里的 lineage / provenance 元数据，未来可能直接决定一份数据集敢不敢卖给模型厂商。56

S1 ④ 日经：美五大科技巨头表外债务激增至 1.65 万亿美元；Oracle 信用风险创 18 年新高

来源：日经（经 CNMO 转引）/ Bloomberg（经 NDTV Profit 转引），07-21

Alphabet、微软、亚马逊、Meta、Oracle 的表外债务四年膨胀 8 倍至 1.65 万亿美元，已超过表内债务总额；五巨头 2026 年 AI 资本开支预计突破 8,000 亿美元（约占美国 GDP 2.5%），2027 年或达 1.1 万亿。同日 Oracle 五年期 CDS 升至约 203bp，创 2008 年以来新高；S&P 本月已将其评级降至距垃圾级仅一档。

点评： AI 军备竞赛的账本正从利润表挪到表外——长期租约、采购承诺、复杂财务安排，市场开始给「看不见的杠杆」定价。Oracle 成了信用市场的 AI 晴雨表，这一幕在 2000 年光纤泡沫里演过。 对你： 写代码的也该关心：云厂商为融资成本焦虑时，第一波传导就是算力涨价与免费额度收紧——DeepSeek 峰谷定价和这条新闻是同一个故事的两端。78

数据与 Lakehouse

S1.5 Apache Amoro 主干推进 AIP-5 Phase 2：优化资源按需弹性扩容（GitHub 提交，07-22） dynamic-allocation 分组迎来需求驱动扩容：指数爬坡（1→2→4→8）按实际积压钳制、冷启动单实例探针、注册引导窗口防重复扩容；检测到瓶颈在规划（planning）而非线程时只告警不盲扩。 点评： 表优化资源的 K8s 化调度是 Amoro 相对纯 catalog 方案最实的差异化。Phase 2 把「扩」做细了，接下来看 Phase 3 怎么「缩」——缩容抖动才是生产事故之源。 对你： 这套 demand-driven 思路（backlog 超时 + 指数爬坡 + 配额感知）可直接借鉴到湖上任意任务调度器。

S2 LakeSoul：Rust 向量索引独立构建 + ANN 检索，升级 DataFusion 54（GitHub 提交，07-20） 新增 lakesoul-vector：IVF + RaBitQ 向量索引可离线独立构建，提供 Python API 端到端测试；元数据切到 Arrow IPC schema，并跟进 Vortex 投影下推。 点评： 国产湖格式把向量检索做成一等公民，AI-native lakehouse 路线与 Lance 遥相呼应，继续盯。

S2 Apache Iceberg 主干：V4 manifest 基础类型进场，远程扫描规划渐成（GitHub，本周） 1.11.0（5 月）后无新版本，但主干在堆两类大活：V4 manifest 的 TrackedFile 等基础类型，以及 REST 远程扫描规划（Spark 侧已可启用）。另：仓库新增 AGENTS.md 与《AI 生成贡献指南》。 点评： 连 Iceberg 都开始给 AI coding agent 立规矩了——那份 Guidelines 本身就值得每个开源维护者读一遍。
